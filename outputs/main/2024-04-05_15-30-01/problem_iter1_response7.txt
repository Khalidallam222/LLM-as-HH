```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead mechanism."""
    # Initialize the best node and its score
    best_node = None
    best_score = np.inf

    # Iterate over all unvisited nodes
    for node in unvisited_nodes:
        # Calculate the direct distance from the current node to the next node
        direct_distance = distance_matrix[current_node][node]
        
        # Look-ahead: estimate the cost of visiting the node 'node' and then returning to the destination
        look_ahead_distance = direct_distance + min(distance_matrix[node][i] for i in unvisited_nodes if i != node)
        
        # Calculate the score as a combination of the direct distance and the look-ahead distance
        score = direct_distance + 0.5 * look_ahead_distance
        
        # Update the best node if the current node has a better score
        if score < best_score:
            best_score = score
            best_node = node

    return best_node
```

This heuristic uses a look-ahead mechanism by considering the direct distance to a node and an estimate of the remaining distance to visit the other nodes, trying to minimize the sum of these two distances. The weight of the look-ahead distance can be adjusted to optimize the heuristic's performance for different problem instances.
