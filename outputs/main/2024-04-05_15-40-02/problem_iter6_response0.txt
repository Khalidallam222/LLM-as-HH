```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead heuristic."""
    # Look-ahead by considering the distance from the next node to all other unvisited nodes
    scores = {}
    for node in unvisited_nodes:
        # Calculate the total distance if we choose this node as the next
        total_distance = distance_matrix[current_node][node]
        remaining_nodes = unvisited_nodes - {node}
        for next_node in remaining_nodes:
            total_distance += distance_matrix[node][next_node]
        # Adjust the score by also considering the distance back to the destination
        total_distance += distance_matrix[next_node][destination_node]
        scores[node] = total_distance
    
    # Select the node with the shortest total distance (least cost)
    next_node = min(scores, key=scores.get)
    return next_node
```

This heuristic function `select_next_node_v2` uses a simple look-ahead strategy to estimate the total cost if the current node is chosen as the next one. It sums up the distances from the current node to the candidate node, from the candidate node to all other unvisited nodes, and finally from the last unvisited node back to the destination node. The node with the least estimated total cost is chosen as the next node to visit.
