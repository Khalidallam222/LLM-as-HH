```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead heuristic."""
    # Assign a score to each unvisited node based on the minimum distance to the current node
    # and the minimum distance from that node to the destination node
    scores = {}
    for node in unvisited_nodes:
        # Look-ahead: choose the node that has the shortest path to the destination
        look_ahead_distance = distance_matrix[node].min() if node != destination_node else np.inf
        # Score is a weighted sum of the direct distance to the node and the look-ahead distance
        score = distance_matrix[current_node][node] + 0.5 * look_ahead_distance
        scores[node] = score

    # Select the node with the lowest score
    next_node = min(scores, key=scores.get)
    return next_node
```
This function uses a look-ahead strategy, considering not only the distance to the next node but also the shortest distance from that node to any other unvisited node, including the final destination. This heuristic aims to prioritize nodes that are likely to lead to a shorter overall path.
