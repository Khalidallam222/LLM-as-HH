```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead heuristic."""
    # Look-ahead by considering the distance from the next node to the remaining unvisited nodes
    scores = {}
    for node in unvisited_nodes:
        # Calculate the total distance if we choose this node as the next
        look_ahead_distance = sum(distance_matrix[node][n] for n in unvisited_nodes if n != node)
        # Score is based on the distance to the current node, the look-ahead distance, and the distance to the final destination
        score = distance_matrix[current_node][node] + look_ahead_distance - distance_matrix[destination_node][node]
        scores[node] = score
    
    # Select the node with the lowest score (least total distance)
    next_node = min(scores, key=scores.get)
    return next_node
```
This function `select_next_node_v2` incorporates a look-ahead heuristic by considering the sum of distances from the candidate next node to all the remaining unvisited nodes, thus trying to minimize the total path length.
