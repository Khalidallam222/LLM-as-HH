```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead heuristic."""
    # Look-ahead by considering the distance from the potential next node to the remaining unvisited nodes
    def look_ahead_score(node, unvisited):
        return sum(distance_matrix[node][n] for n in unvisited if n != node)
    
    scores = {}
    for node in unvisited_nodes:
        # Combine current distance with a look-ahead score
        current_distance = distance_matrix[current_node][node]
        look_ahead = look_ahead_score(node, unvisited_nodes)
        # Heuristic: Prioritize lower current distance and lower look-ahead score
        scores[node] = current_distance + look_ahead
    
    # Select the node with the lowest combined score
    next_node = min(scores, key=scores.get)
    return next_node
```
This function uses a look-ahead heuristic to score each unvisited node by considering the sum of distances from that node to all the remaining unvisited nodes. The node with the lowest combined score (current distance + look-ahead score) is chosen as the next node to visit.
