```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead heuristic."""
    # Look-ahead heuristic: consider the distance to the next node and the subsequent node
    scores = {}
    for node in unvisited_nodes:
        # Evaluate the cost of going to the current node and then to the next unvisited node
        look_ahead_score = min(distance_matrix[node][n] for n in unvisited_nodes if n != node)
        # Combine the current distance with a look-ahead score
        score = distance_matrix[current_node][node] + 0.5 * look_ahead_score - 0.2 * distance_matrix[destination_node][node]
        scores[node] = score

    # Select the node with the lowest score (best heuristic)
    next_node = min(scores, key=scores.get)
    return next_node
```
This function uses a look-ahead mechanism by considering the minimum distance from the potential next node to any of the remaining unvisited nodes, which helps to avoid locally optimal but globally suboptimal paths. The weights (0.5 and 0.2) can be fine-tuned to adjust the heuristic's behavior.
