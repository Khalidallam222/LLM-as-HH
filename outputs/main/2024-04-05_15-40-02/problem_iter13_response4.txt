```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead heuristic."""
    # Look-ahead by considering the distance from the potential next node to the remaining unvisited nodes
    scores = {}
    for node in unvisited_nodes:
        # Calculate the sum of distances from the current node to the potential next node and from that node to all others
        look_ahead_distance = sum(distance_matrix[node][n] for n in unvisited_nodes if n != node)
        # Combine this with the direct distance to the potential next node and the distance from that node to the final destination
        scores[node] = distance_matrix[current_node][node] + look_ahead_distance + distance_matrix[node][destination_node]
    
    # Select the node with the shortest cumulative distance (least score)
    next_node = min(scores, key=scores.get)
    return next_node
```
This function `select_next_node_v2` looks ahead by considering the total distance from the current node to the potential next node and then from that next node to all other unvisited nodes, including the final destination. It tries to minimize the cumulative distance that would result from choosing that next node.
