```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """
    Select the next node to visit from the unvisited nodes using a look-ahead heuristic.
    """
    # Look-ahead: consider the distance from the potential next node to the remaining nodes
    look_ahead_scores = {}
    for node in unvisited_nodes:
        # Calculate the sum of distances from this node to all other unvisited nodes
        look_ahead_distance = sum(distance_matrix[node][n] for n in unvisited_nodes if n != node)
        # Use the inverse of the distance as a score (smaller distance = higher score)
        look_ahead_scores[node] = -look_ahead_distance
    
    # Also consider the direct distance to the current node and the potential gain of going to the destination next
    final_scores = {}
    for node in unvisited_nodes:
        final_scores[node] = look_ahead_scores[node] - distance_matrix[current_node][node] + distance_matrix[node][destination_node]
    
    # Select the node with the highest final score
    next_node = max(final_scores, key=final_scores.get)
    return next_node
```
This function `select_next_node_v2` uses a look-ahead heuristic by calculating the sum of distances from each potential next node to all other unvisited nodes. It then combines this with the direct distance from the current node to the potential next node and from the potential next node to the destination node to compute a final score for each node. The node with the highest final score is chosen as the next node to visit.
