```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead heuristic."""
    # Look-ahead by considering the distance from the next node to the remaining unvisited nodes
    scores = {}
    for node in unvisited_nodes:
        # Calculate the sum of distances from the current node to the next node and from the next node to all other unvisited nodes
        total_distance = distance_matrix[current_node][node] + np.sum(distance_matrix[node][list(unvisited_nodes - {node})])
        # Use the total distance as the score for the node
        scores[node] = total_distance
    
    # Select the node with the shortest total distance (best score)
    next_node = min(scores, key=scores.get)
    return next_node
```
This heuristic function looks ahead and calculates the total distance from the current node to the potential next node and then from that next node to all other unvisited nodes. It then selects the node with the shortest total distance, which should lead to a better overall path.
