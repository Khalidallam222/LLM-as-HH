```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead heuristic."""
    # Define weights for the scoring heuristic
    c1, c2 = 0.7, 0.3
    
    # Initialize the dictionary to store scores for each node
    scores = {}
    
    # Iterate through the unvisited nodes to calculate a score
    for node in unvisited_nodes:
        # Look-ahead: calculate the distance from the current node to the next node and then to the destination
        look_ahead_distance = distance_matrix[current_node][node] + distance_matrix[node][destination_node]
        
        # Calculate the score as a weighted sum of direct distance and look-ahead distance
        score = c1 * distance_matrix[current_node][node] + c2 * look_ahead_distance
        
        # Save the score in the scores dictionary
        scores[node] = score
    
    # Select the node with the minimum score as the next node
    next_node = min(scores, key=scores.get)
    
    return next_node
```
This function uses a simple look-ahead heuristic by considering both the direct distance from the current node to the next node and the cumulative distance from the current node through the next node to the destination node. The weights `c1` and `c2` can be adjusted to emphasize one factor over the other.
