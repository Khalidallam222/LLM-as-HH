```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead heuristic."""
    # Look-ahead heuristic with two criteria: minimum distance to the next node and minimum distance from the next node to the destination
    next_node_scores = {}
    for node in unvisited_nodes:
        # Score based on the sum of the distance from current to the next node and from the next node to the destination
        score = distance_matrix[current_node][node] + distance_matrix[node][destination_node]
        next_node_scores[node] = score
    
    # In case of ties, choose the node with the minimum distance to the final destination
    min_score = min(next_node_scores.values())
    best_nodes = [node for node, score in next_node_scores.items() if score == min_score]
    return min(best_nodes, key=lambda x: distance_matrix[x][destination_node])
```
This function `select_next_node_v2` looks ahead by considering the sum of distances from the current node to the potential next node and from that next node to the final destination. It handles ties by selecting the node that is closest to the final destination.
