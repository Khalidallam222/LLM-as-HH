```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead heuristic."""
    # Assign a score to each unvisited node based on the minimum distance to the remaining nodes
    scores = {}
    for node in unvisited_nodes:
        # Calculate the sum of distances from the current node to the unvisited node and from the unvisited node to the destination
        look_ahead_distance = sum(distance_matrix[node][n] for n in unvisited_nodes if n != node)
        # The score is based on the direct distance to the node and the look-ahead distance
        scores[node] = distance_matrix[current_node][node] + look_ahead_distance
    
    # Select the node with the minimum score (shortest total distance)
    next_node = min(scores, key=scores.get)
    return next_node
```
This function uses a simple look-ahead heuristic by summing the distances from the current node to the unvisited node and from that unvisited node to all other unvisited nodes. This is a very basic approach and can be improved further, but it serves as a starting point.
