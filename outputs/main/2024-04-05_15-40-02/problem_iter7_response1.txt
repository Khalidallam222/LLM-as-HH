```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead heuristic."""
    # Look-ahead by considering the distance from the potential next node to the remaining unvisited nodes
    scores = {}
    for node in unvisited_nodes:
        # Compute the total distance if we choose this node as the next
        look_ahead_distance = sum(distance_matrix[node][n] for n in unvisited_nodes if n != node)
        # Heuristic score considering the direct distance to the current node and the look-ahead distance
        scores[node] = distance_matrix[current_node][node] + look_ahead_distance
    
    # Find the node with the minimum heuristic score
    next_node = min(scores, key=scores.get)
    return next_node
```

This function uses a simple look-ahead heuristic by summing the distances from a potential next node to all other unvisited nodes, then adds this sum to the distance from the current node to the potential next node. The node with the smallest combined score is chosen as the next node to visit.
