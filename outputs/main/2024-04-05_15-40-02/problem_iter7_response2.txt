```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead heuristic."""
    # Look-ahead by considering the distance from the next node to the remaining nodes
    scores = {}
    for node in unvisited_nodes:
        # Score based on the direct distance from the current node to the next node
        direct_distance = distance_matrix[current_node][node]
        # Look-ahead score: sum of distances from the next node to all other unvisited nodes
        look_ahead_distance = sum(distance_matrix[node][i] for i in unvisited_nodes if i != node)
        # Heuristic score is a weighted sum of direct and look-ahead distances
        scores[node] = direct_distance + 0.5 * look_ahead_distance  # 0.5 is a weight that can be adjusted
        
    # Select the node with the lowest heuristic score
    next_node = min(scores, key=scores.get)
    return next_node
```
This `select_next_node_v2` function uses a simple look-ahead heuristic that takes into account the sum of distances from the next node to all other unvisited nodes, in addition to the direct distance from the current node to the next node. The weight of the look-ahead distance can be adjusted to balance the influence of direct and look-ahead distances on the heuristic score.
