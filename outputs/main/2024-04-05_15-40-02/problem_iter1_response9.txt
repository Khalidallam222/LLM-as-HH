```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead mechanism."""
    # Define weights for the scoring heuristic
    c1, c2, c3 = 0.5, 0.3, 0.2
    scores = {}
    
    for node in unvisited_nodes:
        # Calculate the direct distance from the current node to the potential next node
        direct_distance = distance_matrix[current_node][node]
        
        # Calculate the minimum distance from the potential next node to all other unvisited nodes
        min_distance_to_others = min(distance_matrix[node][i] for i in unvisited_nodes if i != node)
        
        # Calculate the distance from the potential next node to the destination node
        distance_to_destination = distance_matrix[node][destination_node]
        
        # Combine the above factors to calculate a score for each potential next node
        score = c1 * direct_distance + c2 * min_distance_to_others + c3 * distance_to_destination
        
        scores[node] = score
    
    # Select the node with the lowest score as the next node
    next_node = min(scores, key=scores.get)
    
    return next_node
```
This `select_next_node_v2` function uses a look-ahead mechanism by considering the minimum distance from the potential next node to all other unvisited nodes. This helps to avoid getting trapped in local optima and encourages the exploration of paths that could lead to a globally better solution.
