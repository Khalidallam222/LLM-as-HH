```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """
    Select the next node to visit from the unvisited nodes using a look-ahead heuristic.
    """
    # Look-ahead heuristic: choose the node that minimizes the total distance to the remaining nodes
    min_distance = float('inf')
    next_node = None
    for node in unvisited_nodes:
        # Calculate the look-ahead distance: sum of distances from current node to the next node and from that node to the destination
        look_ahead_distance = distance_matrix[current_node][node] + distance_matrix[node][destination_node]
        # Find the minimum look-ahead distance
        if look_ahead_distance < min_distance:
            min_distance = look_ahead_distance
            next_node = node
    return next_node
```
This function uses a simple look-ahead heuristic that sums the distances from the current node to a potential next node and from that next node to the final destination. The next node with the smallest total look-ahead distance is selected.
