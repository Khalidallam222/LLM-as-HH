```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead heuristic."""
    # Look-ahead by considering the distance from the potential next node to the remaining unvisited nodes
    scores = {}
    for node in unvisited_nodes:
        # Calculate the sum of distances from the potential next node to all other unvisited nodes
        total_distance_from_node = sum(distance_matrix[node][n] for n in unvisited_nodes if n != node)
        # Score the node based on the distance to the current node, the total distance from the node, and the distance to the final destination
        score = distance_matrix[current_node][node] + 0.5 * total_distance_from_node - 0.3 * distance_matrix[destination_node][node]
        scores[node] = score
    # Select the node with the minimum score (the one that seems to be the most promising)
    next_node = min(scores, key=scores.get)
    return next_node
```
This `select_next_node_v2` function uses a simplified look-ahead heuristic by considering the sum of distances from each potential next node to all other unvisited nodes, along with the distance from the current node and to the final destination. The constants used for scoring can be adjusted to fine-tune the heuristic performance.
