```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """
    Select the next node to visit from the unvisited nodes using a look-ahead heuristic.
    """
    # Look-ahead heuristic: consider the total distance to the next node and the subsequent node
    best_score = float('inf')
    best_node = None
    for node in unvisited_nodes:
        # Score based on the direct distance to the next node and the distance from the next node to the destination
        score = distance_matrix[current_node][node] + distance_matrix[node][destination_node]
        # Optionally, you can add more complexity to the heuristic by considering other factors
        # such as the distances from the potential next node to all other unvisited nodes.
        
        # Update the best node based on the score
        if score < best_score:
            best_score = score
            best_node = node
            
    return best_node
```

This `select_next_node_v2` function is a simplified look-ahead heuristic that tries to minimize the sum of the distances from the current node to the next node and from the next node to the final destination, which is a common approach in solving TSP heuristically. It can be further improved by considering more complex heuristics that take into account the overall structure of the remaining graph.
