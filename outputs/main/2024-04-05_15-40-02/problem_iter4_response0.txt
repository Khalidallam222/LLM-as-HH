```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """
    Select the next node to visit from the unvisited nodes using a look-ahead heuristic.
    """
    # Look-ahead heuristic: choose the node that minimizes the sum of the current distance
    # and the estimated distance to the remaining nodes (using a simple average).
    scores = {}
    for node in unvisited_nodes:
        # Calculate the total distance if we choose this node next
        remaining_nodes = unvisited_nodes - {node}
        total_distance = distance_matrix[current_node][node]
        # Estimate the remaining distance by summing distances from the current node to all remaining nodes
        # and dividing by the number of remaining nodes (excluding the destination node)
        remaining_distance_estimate = sum(distance_matrix[node][n] for n in remaining_nodes) / len(remaining_nodes)
        # Penalize the score if the node is far from the final destination
        distance_to_destination = distance_matrix[node][destination_node]
        score = total_distance + remaining_distance_estimate - distance_to_destination
        scores[node] = score
    
    # Select the node with the smallest score (the one that seems to minimize the total path length)
    next_node = min(scores, key=scores.get)
    return next_node
```
This heuristic function `select_next_node_v2` tries to minimize the total path length by looking ahead and estimating the remaining distance from each potential next node to all the other unvisited nodes. It then adds this estimate to the current distance to that potential next node. The node with the smallest score is selected as the next node to visit.
