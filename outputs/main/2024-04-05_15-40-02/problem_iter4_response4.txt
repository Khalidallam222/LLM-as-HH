```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """
    Select the next node to visit from the unvisited nodes using a look-ahead heuristic.
    """
    # Look-ahead heuristic: minimum spanning tree from the current node to all unvisited nodes
    # combined with the distance to the final destination.
    scores = {}
    for node in unvisited_nodes:
        # Calculate the distance to the node and the minimum distance from that node to any remaining unvisited node
        distance_to_node = distance_matrix[current_node][node]
        distances_from_node = [distance_matrix[node][n] for n in unvisited_nodes if n != node]
        min_distance_from_node = min(distances_from_node) if distances_from_node else 0
        # Score is based on the distance to the node, the minimum distance from the node,
        # and the distance from that node to the final destination
        score = distance_to_node + min_distance_from_node - distance_matrix[node][destination_node]
        scores[node] = score

    # Select the node with the minimum score as the next node to visit
    next_node = min(scores, key=scores.get)
    return next_node
```
This `select_next_node_v2` function uses a look-ahead strategy that considers the minimum distance from the current node to a potential next node and from that next node to the final destination, trying to minimize the overall path length.
