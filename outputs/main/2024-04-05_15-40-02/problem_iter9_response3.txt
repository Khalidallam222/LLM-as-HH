```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead heuristic."""
    # Look-ahead by considering the distance from the next node to the remaining unvisited nodes
    scores = {}
    for node in unvisited_nodes:
        # Calculate the sum of distances from the current node to the next node and from the next node to all other unvisited nodes
        total_distance = distance_matrix[current_node][node] + np.sum([distance_matrix[node][n] for n in unvisited_nodes if n != node])
        # Penalize if the next node brings us closer to the destination node (to avoid ending the tour too soon)
        lookahead_penalty = distance_matrix[node][destination_node] ** 2
        scores[node] = total_distance + lookahead_penalty

    # Select the next node with the lowest total score
    next_node = min(scores, key=scores.get)
    return next_node
```
This `select_next_node_v2` function uses a look-ahead heuristic by considering the sum of distances from the current node to the next node and from the next node to all other unvisited nodes, and it penalizes if the next node is too close to the destination node to encourage a more balanced exploration of the nodes.
