```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead heuristic."""
    # Look-ahead heuristic: choose the node that minimizes the sum of the current distance
    # and the estimated distance to the remaining nodes.
    min_distance = float('inf')
    next_node = -1
    for node in unvisited_nodes:
        # Calculate the current distance to the potential next node.
        current_dist = distance_matrix[current_node][node]
        # Calculate the minimum distance from this potential next node to all other unvisited nodes.
        remaining_dist = min(distance_matrix[node][i] for i in unvisited_nodes if i != node)
        # Combine the current distance with the look-ahead distance.
        total_dist = current_dist + remaining_dist
        # Select the node with the minimum combined distance.
        if total_dist < min_distance:
            min_distance = total_dist
            next_node = node
    return next_node
```
This heuristic function `select_next_node_v2` uses a simple look-ahead strategy, where it tries to minimize the sum of the distance to the next node and the minimum distance from that node to all other unvisited nodes. This approach assumes that minimizing the immediate and future steps can lead to a globally optimal or near-optimal solution for the TSP.
