```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead heuristic."""
    # Look-ahead by considering the distance from the potential next node to the remaining unvisited nodes
    scores = {}
    for node in unvisited_nodes:
        # Compute the sum of distances from the current node to the potential next node and from that node to all other unvisited nodes
        total_distance = distance_matrix[current_node][node] + np.sum(distance_matrix[node][list(unvisited_nodes - {node})])
        # Use the total distance as a score (lower is better)
        scores[node] = total_distance
    
    # Select the node with the lowest total distance (score)
    next_node = min(scores, key=scores.get)
    return next_node
```
This function uses a simple look-ahead heuristic that sums the distances from the current node to the potential next node and from that next node to all the remaining unvisited nodes. This sum is used as a score to determine the next node to visit, with the goal of minimizing the total path length.
