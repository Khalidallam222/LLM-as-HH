```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead heuristic."""
    # Look-ahead by considering the distance from the potential next node to all other unvisited nodes
    scores = {}
    for node in unvisited_nodes:
        # Calculate the total distance if we choose this node as the next
        look_ahead_distance = sum(distance_matrix[node][n] for n in unvisited_nodes if n != node)
        # Heuristic score is based on the distance to the current node, look-ahead distance, and distance to the final destination
        scores[node] = distance_matrix[current_node][node] + 0.5 * look_ahead_distance + distance_matrix[node][destination_node]
    
    # Choose the node with the lowest heuristic score
    next_node = min(scores, key=scores.get)
    return next_node
```
This heuristic function `select_next_node_v2` tries to minimize the sum of distances from the current node to the next node, the total look-ahead distance from the next node to all unvisited nodes, and the distance from the next node to the final destination. The look-ahead distance is included as a heuristic to try to minimize future distances as well.
