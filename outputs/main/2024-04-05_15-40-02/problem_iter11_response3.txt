```python
import numpy as np

def select_next_node_v2(current_node: int, destination_node: int, unvisited_nodes: set, distance_matrix: np.ndarray) -> int:
    """Select the next node to visit from the unvisited nodes using a look-ahead heuristic."""
    # Look-ahead heuristic: minimize the sum of the current distance and the next potential distance
    scores = {}
    for node in unvisited_nodes:
        # Calculate the total distance if we choose this node next
        total_distance = distance_matrix[current_node][node] + min([distance_matrix[node][next_node] for next_node in unvisited_nodes if next_node != node])
        scores[node] = total_distance
    
    # Select the node with the smallest total score (which is the shortest path estimate)
    next_node = min(scores, key=scores.get)
    return next_node
```
This heuristic looks ahead by considering not only the distance from the current node to the potential next node but also the distance from that potential next node to the remaining unvisited nodes. It attempts to minimize the sum of these two distances for each potential next node.
